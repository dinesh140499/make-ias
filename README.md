# Make IAS – Website

Website for **Make IAS**, a learning platform for IAS/UPSC aspirants. Built with React, Vite, and Tailwind CSS.

**Live site:** [https://makeias.iqsetters.com/](https://makeias.iqsetters.com/)
**Dummy Site:** [https://dinesh140499.github.io/make-ias/](https://dinesh140499.github.io/make-ias/)

---

## Tech stack

- **React** + **Vite**
- **Tailwind CSS** (with `@tailwindcss/vite`)
- **React Router**
- **Redux Toolkit** + **Redux Saga**
- **Formik** + **Yup** (forms & validation)
- **ApexCharts**, **Swiper**, **React Icons**, and other UI libraries

---

## Getting started

### Prerequisites

- Node.js (v18 or later)
- npm

### Install dependencies

```bash
npm install
```

### Run locally (development)

```bash
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) (or the URL shown in the terminal).

### Preview production build locally

```bash
npm run build
npm run preview
```

Opens the built site at [http://localhost:4173/make-ias/](http://localhost:4173/make-ias/).

---

## Build & deploy (GitHub Pages)

1. **Build and deploy**

   ```bash
   npm run deploy
   ```

   This runs `npm run build` and then publishes the `dist` folder to the `gh-pages` branch.

2. **GitHub repo settings**

   - Repo → **Settings** → **Pages**
   - **Source:** Deploy from a branch
   - **Branch:** `gh-pages` → **/ (root)** → Save

3. **Live URL**

   After a minute or two, the site is available at:

   **https://dinesh140499.github.io/make-ias/**

## Scripts

| Command           | Description                          |
|-------------------|--------------------------------------|
| `npm run dev`     | Start Vite dev server                |
| `npm run build`   | Build for production                 |
